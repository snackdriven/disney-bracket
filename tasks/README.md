# Tasks

This directory tracks open and completed tasks for the disney-bracket project.

## Structure

- `open/` — tasks that are pending or in progress
- `closed/` — tasks that have been completed

## Format

Each task is a markdown file named `NNN-short-description.md` with the following front matter:

```
---
title: Short task title
status: open | in-progress | closed
created: YYYY-MM-DD
---
```
